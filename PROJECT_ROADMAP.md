# Helm Finance AI CFO — Project Roadmap

## Vision

Build the financial operating system for small businesses and startups.

Users should be able to:

* connect bank accounts
* connect payment processors
* upload invoices
* upload receipts
* manage employees
* track payroll
* monitor cash flow
* forecast runway
* manage accounts payable
* manage accounts receivable
* receive AI CFO recommendations

The system must eventually operate as an autonomous AI CFO assistant.
