Not startup MVP.
Not accounting software.

- Stripe
- Ramp
- Mercury
- Linear
- Arc Browser

---

## UX Rules

Every screen must answer:

1. What is happening?
2. What needs attention?
3. What should I do next?

If a component does not help answer these questions, remove it.
